# Game Market Analysis Project Plan

## Overview

This analysis was designed to explore the commercial performance of console and PC video games between 2018 and 2025. Using SQL, Python and Power BI, the project investigates market trends, pricing, platform strategy and review performance to identify comparable games and support commercial decision-making in the games industry through benchmarking, scenario analysis and forecasting.

## Problem

Game publishers regularly need to estimate the commercial performance of upcoming releases, and this can be done by looking at previous evidence in the games market.

This project aims to investigate how historical market data can be used to

- identify comparable games
- analyse market trends
- understand factors affecting commercial performance
- support scenario analysis
- provide forecasts for hypothetical future titles

## Questions

The project aims to answer questions such as:

- Which characteristics best predict commercial success?
- Which games are the strongest benchmarks for comparison?
- How do genre, platform and pricing influence sales performance?
- How does review reception relate to commercial performance?
- What impact might releasing on an additional platform have?
- How could launch timing affect expected performance?

## Objectives

- Build a relational SQL database.
- Perform exploratory data analysis using Python.
- Identify benchmark games.
- Investigate factors influencing commercial performance.
- Present findings through an interactive Power BI dashboard.
